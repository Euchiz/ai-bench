# qPCR ddCT Analyzer examples

## Files

- `mock_qpcr_input.tsv`: a compact example showing the column layout the GUI expects in an Excel export.
- `example_template.json`: an example saved configuration template.

## Suggested usage

1. Use `mock_qpcr_input.tsv` as a reference to structure your own Excel sheet.
2. Export or save your real qPCR data as `.xlsx`.
3. Launch `python qpcr_analyzer_gui.py`.
4. Load your Excel file and, if useful, load `example_template.json` as a starting point.
